netviz Examples

Network visualization library with force-directed graphs, edge bundling, grouping layouts, and reactive widgets integration.

Note: Run these examples from a web server: npm run serve from the project root.
  • Basic Example — Simple force-directed graph
  • Advanced Features — Edge bundling, Force-in-a-Box, Smart labels
  • Observable Compatibility — Invalidation, state preservation, updates

API Reference

const graph = ForceGraph(data, {
  width: 960,
  height: 600,
  renderer: "canvas",     // or "svg"
  nodeId: d => d.id,
  nodeGroup: d => d.group,
  nodeRadius: 5,
  useEdgeBundling: false,
  useForceInABox: false,
  useSmartLabels: true,
});

document.body.appendChild(graph);
