nodeRadius

Sets the radius of each node in pixels. Can be a fixed number for uniform sizing, or a function to compute size dynamically based on node properties (e.g., degree, weight, importance).

Type number | function
Default 3
Fixed Radius: 5px
ForceGraph(data, { nodeRadius: 5 })
Dynamic Radius by Degree
ForceGraph(data, { nodeRadius: (d) => { const degree = links.filter( l => l.source === d.id || l.target === d.id ).length; return 3 + degree * 0.5; } })
Interactive: Adjust Radius
Node Radius: 5
Proportional to Value
ForceGraph(data, { nodeRadius: (d) => d.value ? Math.sqrt(d.value) * 2 : 5 })

Common Use Cases

Size by Degree (Connectivity)

Make highly connected nodes larger to emphasize network hubs

Size by Importance/Weight

Use node properties like PageRank, centrality, or custom weights to determine size

Categorical Sizing

Different sizes for different node types (e.g., large for organizations, small for individuals)

nodeGroup

Function that returns a group identifier for each node, used for coloring. Nodes with the same group value will receive the same color from the color scheme.

Type function | undefined
Default undefined
No Grouping (Single Color)
ForceGraph(data, { // nodeGroup not specified // All nodes use default color })
Group by Category
ForceGraph(data, { nodeGroup: (d) => d.group, // Uses d3.schemeTableau10 by default })

Common Use Cases

Community Detection

Color nodes by detected communities or clusters in the network

Type/Category

Distinguish different types of entities (users, products, locations, etc.)

Hierarchical Levels

Show different levels in organizational or taxonomic hierarchies

nodeLabel

Function that returns the label text for each node. Used by smart labels (Voronoi-based placement) or simple labels. Also sets the tooltip/title attribute for nodes.

Type function | undefined
Default undefined
Show Node IDs
ForceGraph(data, { nodeLabel: (d) => d.id, useSmartLabels: true })
Custom Label Format
ForceGraph(data, { nodeLabel: (d) => `${d.name} (${d.group})`, useSmartLabels: true })

nodeStroke, nodeStrokeWidth, nodeStrokeOpacity

Control the outline/border of nodes. nodeStroke sets the color, nodeStrokeWidth sets the thickness, and nodeStrokeOpacity sets transparency.

Type string/function, number, number
Defaults "#fff", 1.5, 1
Thick White Stroke
ForceGraph(data, { nodeStroke: "#fff", nodeStrokeWidth: 3, nodeStrokeOpacity: 1 })
Interactive Stroke Controls
Stroke Width: 1.5
Stroke Opacity: 1
